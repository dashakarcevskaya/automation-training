Сайт: https://gettransfer.com/en

Тест-кейс №1 :
    При правильно введенных логине и пароле пользователь пользователь переходит в его личный кабинет.
    Предусловия : пользователь находится на странице сайта https://gettransfer.com/en.
    Шаги : 
    1. Пользователь вводит существующий логин в поле Email/Phone.
    2. Пользователь вводит пароль.
    Результат : Пользователь попадает в личный кабинет, где отображаются его заявки.

Тест-кейс №2 :
    Нельзя зарегистрироваться на сайте не введя номер телефона в форме Sign up (Enter phone number).
    Предусловия : пользаватель находится на сайте в окне формы регистрации (https://gettransfer.com/en/account/new).
    Шаги :
    1. Пользователь заполняет все поля кроме Enter phone number.
    2. Нажимает кнопку Sign up.
    Результат : Подсвечивается поле Enter phone number и выдает сообщение Please fill out this field.

Тест-кейс №3 :
    Нельзя арендовать машину не выбрав город откуда/куда выезд/въезд.
    Предусловия : пользаватель находится на главной странице сайта  https://gettransfer.com/en.
    Шаги :
    1. Пользователь заполняет все поля кроме From.
    2. Нажимает кнопку Get offers.
    Результат : Подсвечивается поле From и выдает сообщение Please fill out this field.

Тест-кейс №4 :
    Нельзя вернуть автомобиль раньше времени начала аренды .
    Предусловия : пользаватель находится на странице сайта https://gettransfer.com/en/transfers/new для добавления информации о поездке.
    Шаги :
    1. Пользователь заполняет Transfer date 2019-11-12 и Pick-up time 15:30.
    2. Пользователь заполняет Return transfer date 2019-11-12 и Pick-up time 14:30.
    3. Нажимает кнопку Get Offers.
    Результат : На странице появляется сообщение Date_return is earlier than departure date.

Тест-кейс №5 :
    Нельзя авторизоваться на сайте, вводя e-mail и пароль, которых нет в системе.   
    Предусловия : пользователь находится на странице https://gettransfer.com/en/account/login.
    Шаги :
    1. Пользователь вводит недействительный email "abcdefg@mail.ru" и пароль "12345678"
    2. Пользователь нажимает кнопку Log in.
    Результат : На странице появляется сообщение Account not found.

Тест-кейс №7 :
    После завершения бронирования машины в личном кабинете пользователя появляется его заказ.   
    Предусловия : пользователь находится в личном кабинете.
    Шаги :
    1. Пользователь выбирает место отправления и пункт назвачения, машину, доп. параметры.
    2. Пользователь завершает заказ.
    Результат : В личном кабинете появилась заявка на машину
